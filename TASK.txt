Вимоги:
1. Завдання має бути виконане на ванільному JavaScript без використання JavaScript бібліотек та фреймворків.
2. При виконанні завдання дозволяється додавати id, користувацькі атрибути елементам
3. Код має бути оформлений в окремий js файл


Завдання:
1. При додаванні товарів реалізувати вивід кількості товарів і загальної ціни в блоці вгорі вікна ( .top-cart-info__item ) у наступному вигляді:
  "Товарів в корзині - XXX, на суму XXX грн"

2. Самостійно розподілити товари по категоріях (по 4 товара в кожну категорію) згідно значень елемента select Категорія
  Реалізувати фільтрацію товарів по категоріях і ціні. Вибір товару повинен працювати по двох фільтрах одночасно.

3. При натисканні на кнопку ОФОРМИТИ ЗАМОВЛЕННЯ вивести модальне вікно з 2-ма полями - Ім'я, email і кнопкою Надіслати
   При настисканні кнопки Надіслати перевірити заповнення полів форми:
   1. Якщо поля не заповнені (або заповнені тільки пробілами) не дозволяти відправку форми, при цьому вивести повідомлення в alert

   2. Якщо поля пройшли валідацію вивести через alert користувачу повідомлення з подякою за покупки,
    очистити верхній блок з інформацією про товари, добавлені в корзину
      "Товарів в корзині - XXX, на суму XXX грн"
